## I. To prepare
```bash
npm install json-server --save-dev
```
package.json
```json
{
  "name": "json-server-tutorial",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "json-server": "json-server --watch data/db.json --port 8000"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "devDependencies": {
    "json-server": "^0.17.0"
  }
}
```

## II. To put in work 
```bash
npm run json-server
```
Right-click on <code>index.html</code> and choose <code>Open with Live Server</code>.
